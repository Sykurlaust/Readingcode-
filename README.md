## Project files

- `level 1 index.html`
  Basic HTML page with one `<h1>` heading.

- `level 2 index.html`
  Shows how a CSS class (`.different`) changes the `<h1>` background color.

- `level 3 style.css`
  CSS comment example and a `.different` class that sets text color to green.

- `level 4 index.html`
  Example using Tailwind-style utility classes in the `class` attribute.

- `level 5 index.html`
  JavaScript example that finds an element by id (`greeting`) and sets:
  `innerHTML = "Hello, world!"`.
